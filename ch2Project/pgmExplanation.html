Explaining a Simple Grade Checker Program

This web page explains a short JS program that decides whether a student passes based on a test score. In this context, JS stands for JavaScript, a language used to add logic to web pages. The idea is simple, but it shows how input, variables, and output work together.

The Code Fragment

Below is the entire program. It is short and focused, which makes it easy to analyze: 


let score = 78;
let passingScore = 70;

if (score >= passingScore) {
  console.log("You passed!");
} else {
  console.log("You did not pass.");
}

How It Works

The specific variable score stores the input value 78. The variable passingScore is set to 70. When the program runs, it compares these two values. If score is greater than or equal to passingScore, the program prints the output message You passed!. Otherwise, it prints You did not pass..

In practice, a user might type a number using the keyboard, such as 78, and the program would evaluate it. The result you see in the console is the programâ€™s final decision.

Why This Matters

As Clean Code by Robert C. Martin explains, Clean code always looks like it was written by someone who cares. This small example shows how even a tiny program can be written clearly and read easily.

Notice how the logic is easy to follow: first the data is stored, then a decision is made, and finally a message is shown. This is a basic pattern you will see in many programs, from small scripts to large applications.

If you changed score to 65, the output would change, which demonstrates how sensitive programs are to their input.
